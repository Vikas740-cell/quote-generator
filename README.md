# 🌟 Lumina | Premium Quote Generator

Lumina is a modern, visually stunning Quote Generator built with **React**, **Tailwind CSS 4**, and **Framer Motion**. It provides a premium user experience with smooth glassmorphism effects, dynamic background animations, and an intelligent theme system.

## ✨ Features

- **Premium Aesthetics**: Sophisticated glassmorphism card with deep adaptive shadows and 3XL backdrop blur.
- **Dynamic Mesh Background**: High-end animated radial mesh gradients that respond to the active theme.
- **Micro-Interactions**: One-click "Copy to Clipboard" with visual confirmation and Twitter/X social sharing.
- **Theme Intelligence**: Beautifully crafted Dark and Light modes with system preference detection.
- **Fluid Animations**: Smooth icon transitions, floating background blobs, and quote fade-effect using Framer Motion.
- **Real-time Wisdom**: Fetches inspirational quotes via Axios from a reliable API with built-in failure safety.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **HTTP Client**: [Axios](https://axios-http.com/)
- **Build Tool**: [Vite 6](https://vitejs.dev/)

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Vikas740-cell/quote-generator.git
   cd quote-generator
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
